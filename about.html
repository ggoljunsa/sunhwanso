이 페이지는 DGIST UI/UX 디자인 강의 2026-1학기 후반기 팀프로젝트 6조의 미드파이 프로토타입입니다.

옷에 달린 NFC 태그에 폰을 대면 그 옷의 이전 사용자들·메시지·수선 기록이 펼쳐집니다. "이 정장은 32명의 디펜스를 함께했어요"처럼 단순한 대여를 넘어 이야기를 잇는 캠퍼스 자원 순환 플랫폼입니다.

핵심 기능 4

  • Browse — 카테고리 + 누적 스토리 미리보기
  • Loop — 신청 → 픽업(NFC) → 사용 → 반납(NFC) → 리워드
  • NFC Story ★ — 옷마다 사용 이력·메시지 누적 (우리만의 차별점)
  • 디지온도 + Map — 반납 신뢰도 온도 + 순환소 위치

NFC 태그 → URL 매핑

  • index.html — 메인 앱 진입 (홈/Browse)
  • loop.html?id=00032 — 5단계 대여 흐름 (정장)
  • story.html?id=00032 — 디펜스 정장 #32 스토리
  • story.html?id=00047 — 장우산 #47 스토리
  • story.html?id=00008 — 실험복 #8 스토리
  • story.html?id=00001 — 블레이저 (첫 사용자)
  • temp.html — 디지온도 + Map
